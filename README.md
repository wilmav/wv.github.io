# Sovellukset.com

Sovellukset.com on projekti, jonka puitteissa kehitän visioista toimivia verkkotyökaluja. Tavoitteenani on rakentaa kevyitä, hyödyllisiä ja teknisesti mielenkiintoisia selainpohjaisia sovelluksia hyödyntäen modernia AI-avusteista kehitystä ja avoimia rajapintoja.

## Kuitulaskuri
Kuitulaskuri on monikielinen työkalu päivittäisen kuitutavoitteen seurantaan. Se on ensimmäinen toteutus tässä projektiympäristössä.

### Projektin tavoitteet
- Tutkia AI-avusteista ohjelmistokehitystä ("Vibe coding").
- Hyödyntää julkisia rajapintoja, esimerkiksi Fineli API.
- Luoda toimivia ja kevyitä verkkotyökaluja.

### Teknologiat
- **Frontend:** Vanilla HTML, CSS ja JavaScript.
- **API:** Fineli (THL) elintarvikkeiden koostumustietokanta.

### Ominaisuudet
- **Monikielinen haku:** suomi, ruotsi ja englanti suoraan API:sta.
- **Päivittäinen seuranta:** kuidun ja kalorien laskenta valittujen annosten perusteella.
